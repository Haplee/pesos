# Comandos útiles para GymLog

## Desarrollo local

```bash
# Vercel dev (recomendado)
vercel dev

# O simple abrir index.html en navegador
```

## Despliegue

```bash
# Desplegar a Vercel
vercel

# Desplegar a producción
vercel --prod
```

## Git

```bash
# Estado
git status

# Añadir cambios
git add .

# Commit
git commit -m "descripción"

# Push a remoto
git push origin main
```
